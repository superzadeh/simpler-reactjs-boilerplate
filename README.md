# Simpler React Boilerplate

## Setting up

```bash
npm install -g yarn
yarn install
```

## Development

```bash
yarn start
```

## Testing

```bash
yarn test
# or
yarn test:watch
```

## What changed from the ReactJS Boilerplate

- Removed ImmutableJS
- Changed File structure
- Support Sass
- Leaner template website
- i18n present but not used by default (use `messages` from `react-intl`)
